# 分布式光伏自动清洁系统 (Distributed PV Auto Cleaner)

本系统旨在解决分布式光伏电站无人化值守过程中光伏组件的清洁问题，构建“空地协同智能清洁体系”。通过引入智能清洁机器人与无人驾驶航空器配合，系统能够实现对光伏组件的自动清洁，从而解决“组件分散、地形复杂、人工清洁成本高、效率低”的痛点，提高电站的发点效率和经济效益。

## 系统组成

1. **智能清洁机器人**：采用履带式运动底盘，配备高压喷淋与旋转毛刷清洁装置，能够自主规划路线、识别污渍并执行清洁任务。
2. **无人驾驶航空器**：电动升降臂转运模块能够运载清洁机器人到达不同光伏组件，解决分布式场景组件分散跨越困难的问题。
3. **中控系统**：基于PC上位机（如QGC）协调机器人与航空器工作，监控清洁过程，设置任务规划，并进行数据分析和报告生成。
4. **图传与遥控系统**：使用专门的图传设备提供实时视频回传，配合 ELRS 控制器为现场操作员提供手动控制与参数调节功能。

## 系统通讯

系统通过无线通信技术结合局域网/广域网实现交互控制：
- **ExpressLRS (ELRS)**：提供低延迟、长距离的双向无线链路，适用于近距离控制和无人机-机器人协同的数据传输。
- **Mavlink 协议**：依赖 ELRS 物理链路，确保无人机飞控与中控之间消息与指令的高效传输。
- **LTE/5G 网络**：支持在广域覆盖下实现远程控制、数据回传和智能调度。
